  1. λ-演算
  • λ-演算 是一套从数学逻辑中发展，以变量绑定和替换的规则，来研究函数如何抽象化定义、函数如何被应用以及递归的形式系统
  • 𝜆-演算 分为 类型化𝜆-演算(Typed λ-Calculus) 和 无类型𝜆-演算(Type-Free λ-Calculus) 也称为 朴素λ-演算(Naive λ-Calculs). 本次主要分享的是无类型的λ-演算
  • 无类型 λ-演算 内核非常小，可以用下面的文法来描述：

$$ \longrightarrow a | b | … \
\longrightarrow λ.<λ-exp> \
\longrightarrow (<λ-exp>) \
<λ-exp> \longrightarrow | | $$

  1. λ-演算中的数字
let 0 = lambda s z . z
let 1 = lambda s z . s z
let 2 = lambda s z . s (s z)
// let n = lambda s z . (n s z)

let add = lambda x y . (lambda s z . x s (y s z))
(add 2 3) = 
    (lambda x y . (lambda s z . x s (y s z))) 2 3  // 展开
    lambda s z . 2 s (3 s z)  // β-规约 右侧函数
     lambda s z . (lambda s2 z2 . s2 (s2 z2)) s (3 s z) // 将数字2展开
    lambda s z . s (s (3 s z)) // β-规约 右侧函数
    lambda s z . s (s ((lambda s3 z3 . s3 (s3 (s3 z3))) s z)) // 将数字3展开
    lambda s z . s (s (s (s (s z)))) // 邱奇数-5
  1. λ-演算中的布尔值与选择
let TRUE = lambda x y . x
let FALSE = lambda x y . y

let And = lambda x y . x y FALSE 
let Or = lambda x y. x TRUE y 
let Not = lambda x . x FALSE TRUE

哈斯凯尔·布鲁克·柯里(Haskell.Brooks.Curry)

  • Y := λf.(λx.(f (x x)) λx.(f (x x))) – Call-By-Name
  • Y := λf.(λu.(u u)) λx.(f λv.(x x v)) – Call-By-Value
// 用一个例子函数g来展开它，可以看到这个函数是如何成为一个不动点组合子的

(Y g)
    = (λf.(λx.(f (x x)) λx.(f (x x))) g) // 展开
    = (λx.(g (x x)) λx.(g (x x))) // λf的β-归约 - 应用主函数于g
    = (λy.(g (y y)) λx.(g (x x)))// α-转换 - 重命名约束变量
    = (g (λx.(g (x x)) λx.(g (x x)))) // λy的β-归约 - 应用左侧函数于右侧函数
    = (g (Y g)) // Y的定义
